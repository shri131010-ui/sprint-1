

To accelerate the development process and resolve styling and responsiveness challenges, I utilized AI assistance (ChatGPT) during this sprint. Below is the log of the prompts used:

### Prompt 1

**Tool Used:** ChatGPT

**Prompt:** "How to implement a dark and light mode theme toggle using HTML, CSS, and JavaScript?"

**Help Received:** Learned how to use CSS variables and JavaScript to switch between dark and light themes by toggling a class on the `<body>` element while maintaining a consistent user experience.

---

### Prompt 2

**Tool Used:** ChatGPT

**Prompt:** "The text and background colors are not properly visible in light mode. How can I fix the contrast issue?"

**Help Received:** Learned how to define separate CSS variables for text, background, and component colors to ensure proper readability and accessibility across both themes.

---

### Prompt 3

**Tool Used:** ChatGPT

**Prompt:** "The service cards blend into the page background. How can I make them stand out visually?"

**Help Received:** Learned how to use CSS properties such as `box-shadow`, `border`, and proper background colors to create depth and visual separation between the cards and the page background.

---

### Prompt 4

**Tool Used:** ChatGPT

**Prompt:** "How to make the website responsive and optimize the mobile view for the navbar and service cards?"

**Help Received:** Learned how to use CSS media queries to adjust layouts for smaller screens, stack service cards vertically, and create a mobile-friendly navigation structure for better usability across devices.
